# Impart-Pro 插件说明
用于模拟牛牛的成长和决斗过程，并提供排行榜等功能。


## 功能列表
| 指令             | 说明                               |
| ---------------- | ---------------------------------- |
| 开导 [@某人]     | 长牛牛                             |
| 决斗 [@某人]     | 战斗！爽~                          |
| 重开牛牛         | 牛牛很差怎么办？稳了！直接重开！   |
| 牛牛排行榜       | 查看牛牛排行榜                     |
| 看看牛牛 [@某人] | 查询自己或者别人牛牛数据           |
| 锁牛牛           | 开启/关闭 某人/某频道 的牛牛大作战 |


## 注意事项
- 确保你已经正确配置了 Koishi 和 Impart-Pro 插件。
- 请根据你的需求调整配置项和数据库表结构。
- 如果遇到问题，请查看插件代码或寻求社区支持。
